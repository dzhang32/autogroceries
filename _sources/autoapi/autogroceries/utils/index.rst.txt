.. py:function:: pause(_func=None, *, delay=1, rand=1)

   Decorator that pauses for a random-ish interval before executing function

   Used to delay the execution of functions that use Selenium by random-ish
   intervals to 1. seem more human-like and 2. avoid being blocked a website
   for too many requests within a set time frame.

   :param _func: Function to be wrapped by decorator.
   :type _func: function, default None
   :param delay: Fixed duration in seconds to pause before function call.
   :type delay: float, default 1
   :param rand: Random time interval to add to fixed delay, generated using
                random.uniform(0, rand).
   :type rand: float, default 1
